### Ship Locator
===========
The project is an exercise of using Play2/Scala/ReactiveMongo/AngularJs/Bootstrap

Based on (https://github.com/lashford/modern-web-template)

#### To run:

clone the project, and execute from the project root:

```
 ./activator clean run
```
